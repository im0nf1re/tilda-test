## Настройка проекта

1.  **Установите зависимости Composer:**
    Используйте локальный исполняемый файл `composer.phar` для установки зависимостей и генерации автозагрузчика:
    ```bash
    php composer.phar dump-autoload
    ```

## Конфигурация

Массив номеров телефонов для городов находится в файле `config/phones.php`

## Запуск приложения

### Использование встроенного веб-сервера PHP

Перейдите в корневую директорию проекта в вашем терминале и выполните:

```bash
php -S 0.0.0.0:8000 -t public/
```

Затем откройте ваш веб-браузер и перейдите по адресу `http://localhost:8000`.

## Тестирование и отладка

*   **Просмотр страницы в обычном режиме:** Откройте `http://localhost:8000`. Отображаемый номер телефона должен соответствовать вашему определенному городу (если настроен).
*   **Имитация IP-адреса:** Вы можете передать IP-адрес в качестве GET-параметра `ip`.
    *   Пример (Екатеринбург): `http://localhost:8000/?ip=94.25.165.48`
    *   Пример (Москва): `http://localhost:8000/?ip=5.250.237.0`
    *   Пример (ненастроенный город/приватный IP): `http://localhost:8000/?ip=10.0.0.1` (должен отобразиться номер 8-800 по умолчанию)
*   **Включение отладочной информации:** Добавьте `&debug` к URL, чтобы увидеть дополнительные детали отладки, включая IP клиента, определенный город и полную карту телефонов.
    *   Пример: `http://localhost:8000/?ip=94.25.165.48&debug`
